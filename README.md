## Getting Started

### Prerequisites

- Node.js 
- npm or yarn

### Installation and Run
- npm install
- npm start

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
